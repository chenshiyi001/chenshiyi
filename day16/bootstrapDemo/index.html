为什么要使用 Grunt？

Grunt 生态系统非常庞大，并且一直在增长。由于拥有数量庞大的插件可供选择，因此，你可以利用 Grunt 自动完成任何事，并且花费最少的代价。如果找不到你所需要的插件，那就自己动手创造一个 Grunt 插件，然后将其发布到 npm 上吧。

在继续学习前，你需要先将Grunt命令行（CLI）安装到全局环境中。

在安装 Grunt 前，请确保当前环境中所安装的 npm 已经是最新版本。
